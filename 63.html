El método DOM:CanvasRenderingContext2D.fillStyle() del API Canvas 2D define el color o estilo a utilizar dentro de las figuras geométricas. Por defecto es negro (#000). El método DOM:CanvasRenderingContext2D.fillRect() del API Canvas 2D dibuja un rectángulo relleno en la posición ( x, y ). El tamaño del rectángulo se determina por width El método DOM:CanvasRenderingContext2D.beginPath() del API Canvas 2D comienza una nueva ruta vaciando la lista de sub-rutas. Invoca este método cuando quieras crear una nueva ruta.
