QUESTION 216 You have the following HTML markup <input name="object1x"> <input name="object1y"> <input name="object2x"> For all of the fields that have names ending with the letter x, you need to set the value to value1. Which code line should you use? A. $("input[name$='x']").html("value1"); B. $("#input[name$='x']").val("value1"); C. $("input[name$='x']").val("value1"); D. $("#input[name$='x']").html("value1");
Respuesta corrrecta C
